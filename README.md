# Marrakech

火星開発をテーマにした、Marrakech風ボードゲームのHTMLデモです。

向き変更 → 移動 → 隣接配置の1ターンを実際に操作できます。

## ゲーム仕様

### 概要

- プレイ人数: 3人（A / B / C）
- 初期所持金: 各30
- 盤面: 六角形マップ（7段, マス数 4-5-6-7-6-5-4）
- 地形タイプ: `sea` / `mountain` / `city`
- 手番進行: A → B → C の順で繰り返し

### 初期状態

- アッサム初期位置: `(3,3)`
- アッサム初期向き: `北東`
- 初期配置済み地形:
  - `(1,1)` sea（B）
  - `(2,1)` sea（B）
  - `(4,2)` mountain（C）
  - `(3,3)` city（A）

### 1ターンの流れ

1. **向き変更フェーズ**
   - アッサム隣接マスをクリックして向きを決定します。
2. **移動フェーズ**
   - 「アッサムを移動」を押すと、1〜3マスをランダム移動します。
   - 進行方向が盤外の場合、盤端を沿う形で折り返し移動します（向きも更新）。
3. **支払い判定**
   - 着地先が「他プレイヤー所有の地形」の場合、着地地形と同タイプ・同オーナーで連結している枚数ぶんを支払います。
   - 支払いは `着地したプレイヤー → 地形オーナー` へ移動します。
4. **配置フェーズ**
   - 1手番で同じ地形タイプを2マス配置します。
   - 1マス目はアッサム隣接マスから選択。
   - 2マス目は1マス目に隣接し、かつアッサム位置ではないマスを選択。
   - 既存地形があるマスにも上書き配置できます。
5. **手番終了**
   - 次プレイヤーに交代し、向き変更フェーズに戻ります。

### 操作UI

- **地形ボタン**: 配置する地形タイプを選択
- **⇢ アッサムを移動**: 移動フェーズを実行
- **イベントログ**: ターンごとの行動、支払い、折り返しなどを表示
- **HUD**: 手番プレイヤー、向き、位置、フェーズ、所持金、配置地形を表示
